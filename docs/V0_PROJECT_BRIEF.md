# VisionGenie 项目描述（用于 V0 高级前端生成）

## 项目PRD描述

### 1. 产品定位
VisionGenie 是一款面向影像创作者的移动端 AI 创作工作台，覆盖从素材处理到内容发布的完整链路。  
产品将「专业图像调色」「2D 转 3D 资产生成」「AI 创作助手」「社区分享复用」「个人创作配置」整合在同一个 App 中，通过全局 Agent 角色串联跨页面任务，形成“创作-优化-发布-复用”的闭环体验。

### 2. 目标用户画像
- 影像内容创作者：需要快速完成调色、风格统一与内容发布。
- 3D 资产探索者：希望从 2D 素材快速生成可用的 3D 模型原型。
- AI 工作流用户：偏好自然语言/语音驱动的创作流程，希望减少参数学习成本。
- 社区型创作者：需要展示作品、获取互动、复用他人调色参数提升效率。

### 3. 信息架构（五 Tab）
底部固定五个一级导航，分别承载完整创作流程：
- 智能调色（`grading`）：图像导入、GPU 实时预览、参数调节、语音调色、导出。
- 2D 转 3D（`convert`）：素材选择、质量策略切换、流程可视化、任务进度反馈。
- AI 助手（`agent`）：能力总览、快捷任务、会话状态、全局操作入口。
- 社区（`community`）：草稿编辑与发布、内容流浏览、点赞收藏评论、参数复用回流调色页。
- 我的（`profile`）：登录注册、资料编辑、工作流偏好设置、账号服务信息。

### 4. 五大页面功能说明与关键链路

#### A. 智能调色工作台（核心生产页）
- 核心模块：
  - 图片输入：相册/相机选择与清空。
  - 实时预览：GPU 前后对比、开关对比模式、Shader 降级提示。
  - 调色能力：预设选择 + 手动参数（基础光效、色彩平衡、曲线、色轮）。
  - 语音调色：按住说话、识别结果、视觉首轮建议、增量叠加、会话撤销。
  - 输出动作：重置参数、导出结果。
- 主链路（必须保留）：
  1. 上传图片
  2. 查看实时预览
  3. 通过预设/滑杆调整参数
  4. 用语音继续精修并可撤销
  5. 导出结果

#### B. 2D 转 3D 工作台
- 核心模块：
  - 输入入口：单图重建 / 多图重建。
  - 重建策略：极速预览 / 平衡模式 / 高精模式。
  - 流程展示：分割与深度、几何重建、网格优化、PBR 材质。
  - 任务状态：任务名、进度条、预计耗时、导出格式。
- 交互节奏：先选输入与质量策略，再启动任务，持续查看阶段与进度反馈。

#### C. AI 助手页 + 全局 Agent 角色
- AI 助手页展示：
  - 能力矩阵（拍摄建议、调色指导、3D 流程编排、发布助手）。
  - 智能会话示例与快捷任务建议。
  - 运行时状态（阶段、待确认动作、历史任务、消息/错误）。
- 全局 Agent（跨页浮动小精灵）：
  - 悬浮球可拖拽，点击展开底部面板。
  - 长按触发语音目标输入。
  - 支持“一键优化当前页”“继续上次任务”。
  - 高风险动作有待确认列表（确认/取消）。
  - 提供最近计划、推理说明、错误反馈、撤销最近自动执行。

#### D. 社区页（发布与复用）
- 核心模块：
  - 草稿编辑：标题、描述、标签。
  - 草稿操作：保存草稿、发布草稿。
  - 社区流：筛选（全部/人像/电影感/复古）、帖子列表。
  - 互动能力：点赞、收藏、评论、回复评论。
  - 参数复用：将帖子调色参数一键回流到调色页继续编辑。
- 主链路（必须保留）：
  1. 创建草稿
  2. 发布到社区
  3. 收到点赞/收藏/评论互动
  4. 他人或自己复用参数返回调色页

#### E. 个人中心（账号与偏好）
- 核心模块：
  - 登录注册：用户名 + 密码，切换登录/注册模式。
  - 资料编辑：昵称、头像 URL、等级文案。
  - 偏好设置开关：
    - Wi-Fi 自动同步任务
    - 社区互动通知
    - 语音命令结束后自动应用
  - 账号信息与统计：当前账号、模型任务数、社区发布数、退出登录。
- 主线目标：让用户可控地管理身份与自动化工作流偏好。

### 5. 页面间关系与状态节奏
- App 级结构：
  - 底部 Tab 是主骨架，所有页面共享统一品牌系统与状态语义。
  - 全局 Agent 悬浮球常驻，作为跨页面任务入口。
- 关键跨页流：
  - 社区“复用参数” -> 调色页立即套用参数并继续编辑。
  - Agent 动作可触发页面切换（如跳到调色页执行优化，跳到社区发布草稿）。
- 状态语义（视觉阶段保留）：
  - 加载态、空态、错误态、成功反馈态。
  - 高风险动作确认态（强调安全与可撤销）。

### 6. 后端能力映射（用于界面语义，不要求工程细节）
- 语音调色解释服务：`/v1/color/interpret`
- Agent 规划与执行服务：`/v1/agent/plan`、`/v1/agent/execute`
- 社区服务：feed、草稿、发布、点赞、收藏、评论
- 账号服务：注册、登录、个人信息、偏好设置
- 设计呈现要求：前端概念稿可展示这些能力入口与状态反馈，但不展开接口实现细节。

## V0主Prompt

为我设计一个移动端优先的高端创作者 App「VisionGenie」，输出完整多页面 UI（不是单页），包含 5 个底部 Tab：智能调色、2D转3D、AI助手、社区、我的。请采用“全新品牌风格”，不要沿用传统蓝黑科技风。整体气质要像国际化创作工具品牌：克制但有辨识度、具备编辑器级专业感与内容平台的活力。  
视觉目标是“概念级高保真”，重点在风格和信息架构，不必实现真实后端逻辑，但要保留关键业务语义标签与状态。  
请统一输出以下页面并保证组件语言一致：  
1) 智能调色页：包含图片导入区（相册/相机）、大尺寸实时预览区（前后对比开关）、预设胶囊列表、参数控制模块（基础光效/色彩平衡/曲线/色轮）、语音调色卡片（识别状态、最近建议、撤销操作）、导出按钮。  
2) 2D转3D页：包含素材输入方式（单图/多图）、质量模式分段控件（极速/平衡/高精）、流程步骤可视化（分割-重建-优化-材质）、任务进度卡（任务名、百分比、预计剩余时间、导出格式）。  
3) AI助手页：包含 Agent 品牌头图、能力矩阵卡片、会话示例气泡、快捷任务列表、运行状态卡片（阶段/待确认动作/历史任务），并提供“打开小精灵面板”主按钮。  
4) 社区页：包含草稿编辑区（标题/描述/标签）、保存与发布按钮、筛选标签条、帖子流卡片（作者/标题/内容/标签/before-after 占位）、互动区（点赞收藏评论）、评论抽屉、复用参数按钮。  
5) 我的页：包含登录注册卡片、用户资料卡、资料编辑表单、工作流偏好开关（Wi-Fi 自动同步/社区通知/语音自动应用）、账号统计、退出登录按钮。  
同时设计全局悬浮 Agent 小精灵：右下角可拖拽圆形浮球，点开后是底部面板，含目标输入框、发送按钮、快捷动作（优化当前页/继续上次任务）、待确认动作列表（确认/取消）、最近状态与撤销入口。  
设计系统要求：  
- 移动端优先（iPhone 14 尺寸基准），并能平滑扩展到平板和桌面。  
- 使用全新品牌叙事：定义一套新的主色、辅色、中性色、语义色，不要蓝黑主导。  
- 强调材质层次：大中小卡片层级、半透明或磨砂材质、柔和阴影、清晰边界。  
- 字体层级清楚：标题、模块标题、正文、元信息、按钮文案有稳定节奏。  
- 图标语义统一：创作、语音、3D、社区、设置等图标风格一致。  
- 动效建议可视化：页面进入、卡片浮现、状态切换、按钮按压反馈（无需写代码，用设计说明体现）。  
- 状态完整：加载、空内容、错误提示、成功反馈、风险操作确认。  
- 输出结果必须是一个完整 App 级设计语言，而不是几个风格不一致的零散页面。
