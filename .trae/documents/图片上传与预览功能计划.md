# 图片上传与预览调色结果功能实施计划

## 一、功能概述

为 VisionGenie 调色功能添加图片上传能力和实时预览调色效果，支持用户从相册选择图片或拍摄照片，并在调色过程中实时查看效果对比。

## 二、实施步骤

### 步骤 1：安装必要的依赖包
- 安装 `react-native-image-picker`：用于图片选择和拍摄
- 安装 `react-native-view-shot`：用于截图保存调色后的图片
- 配置 Android 权限（相机、存储）

### 步骤 2：创建图片选择组件
- 创建 `ImagePicker` 组件
- 支持从相册选择图片
- 支持调用相机拍摄
- 显示已选择图片的缩略图
- 处理图片加载和错误状态

### 步骤 3：实现图片预览与对比功能
- 创建 `BeforeAfterViewer` 组件
- 实现左右分屏对比（原图 vs 调色后）
- 实现滑块拖拽对比效果
- 支持双指缩放查看细节
- 添加对比开关按钮

### 步骤 4：更新调色界面集成图片上传
- 在 `ColorGradingScreen` 中添加图片选择入口
- 显示当前处理的图片
- 图片为空时显示引导界面
- 调色参数实时应用到预览图

### 步骤 5：实现调色结果保存功能
- 添加保存按钮
- 使用 `react-native-view-shot` 捕获调色后的画面
- 保存到相册
- 显示保存成功提示

### 步骤 6：优化用户体验
- 添加图片加载动画
- 优化大图处理性能
- 添加操作引导提示
- 处理边界情况（无权限、图片加载失败等）

## 三、技术要点

### 图片处理
- 使用 React Native 的 Image 组件显示图片
- 通过调整 Image 的 style 实现滤镜效果（后续通过原生模块实现真实调色）
- 图片压缩和缓存策略

### 权限管理
- Android 相机权限
- Android 存储读写权限
- 权限请求和引导

### 性能优化
- 图片懒加载
- 防抖处理参数变更
- 使用 Native 模块处理图片（预留接口）

## 四、预期成果

1. 用户可以点击按钮选择图片或拍摄照片
2. 选中的图片显示在调色界面
3. 可以通过滑动对比原图和调色效果
4. 调色参数实时预览
5. 可以保存调色后的图片到相册
6. 完整的权限处理和错误提示

## 五、文件清单
